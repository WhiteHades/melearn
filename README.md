# melearn

modern local course viewer for offline learning.

## development

```bash
pnpm install
pnpm dev
```

## tauri desktop app

```bash
pnpm tauri dev
pnpm tauri build
```

## tech stack

- next.js 16 (app router)
- react 19
- typescript
- tailwind css 4
- shadcn/ui
- zustand v5
- tauri 2.x (rust)
